Development Log

Features:
- Easy to use, attractive interface
- Framed and Non-Framed Viewing Modes
- Read messages
- Post messages
- Reply to posts
- Member Profiles
- Delete your own messages
- Delete Entire Thread (moderators)
- ICQ Online Status Icon
- Send ICQ message to member
- Message previewing
- Jump directly to specific message (frames mode)
- Simple Search
- IP Logging
- IP Banning
- Administration Mode
- Custom HTML Header Support
- Multiple Forum Support
